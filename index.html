Historia de HTML y CSS

HTML, conocido por las siglas de HyperText Markup Language, es el elemento de construcción más básico de una página web y se usa para crear y representar visualmente una página web.
Su origen se remonta a los años 80, cuando Tim Berners-Lee propuso un nuevo sistema de "hipertexto" para compartir documentos. Tras ser desarrollado por Tim unió fuerzas junto al ingeniero Robert Cailliau para formar una propuesta siendo ganadaora para el internet llamada WorldWideWeb (W3).

Who is the end user?

Al usuario siempre se le considera a aquella persona que será la que vaya a interactuar de manera directa con la página web.

USUARIO FINAL

Who is the client?

El cliente es aquel individuo que busca de nuestros servicios a cambio de dinero para poder realizar algún proyecto, en este caso aquellos que desean obtener una página web.

CLIENTE

Frontend and Backend

El Frontend es aquel que se enfoca en el usurio, en todo lo que puede ver y todo con lo que puede interactuar.
El Backend es el que esta enfocadoent odos los procesos que ocurren detrás para que la página funciones, tomando datos y procesandolos.

FRONT/BACK

CDN architecture

CDN se refuere a Content Delivery Network y su principal función es generar una entrega más rápidad el contenido del sitio, esto lo hace por medio de servidores distribuidos estrategicamente en todo el mundo donde se almacena el caché del contenido.
